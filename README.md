Простые пользователи могут просматривать посты и добавить свой пост на модерацию. 
Администрация - одобрить пост, и выложить его. 
Добавить другие виды админов? Админы могут работать с БД (добавить/удалить пользователя/назначить нового)? Добавить счётчик просмотров на каждом посте?
